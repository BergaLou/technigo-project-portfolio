# My Portfolio Project

## Description

This is my personal portfolio project, built as part of my journey at Technigo. While I based the layout on a provided design, I have fully customized it with my own colors, typography, and personal style to reflect who I am as a developer.

## Tools & Techniques

React: I used this project to practice building functional components and managing a modern frontend workflow.

## CSS & Responsiveness

I focused heavily on making the site mobile-friendly. Even though the responsiveness was a bit tricky at the end, I managed to solve it using Flexbox and Media Queries.

## Vite

Used as the build tool for a fast development experience.

## Challenges & Learning

The most fun part was learning new React concepts! It's a simple setup but still feels very fresh and powerful. Dealing with the CSS layout in the footer and hero section taught me a lot about how fixed units (like rem) can affect mobile views.

## What's Next?

Custom Domain: I am planning to connect a custom domain (bergalou.com or bergalou.se) to make the site even more personal.

Adding more complex JavaScript and Backend projects to the carousel.
